# TAC Token List

This repository contains the canonical Token List for TAC Mainnet

## Quick Start

```bash
# FROM: ./
pnpm install;
pnpm validate;
```

## Overview

- [`src/tokens/`](/src/tokens): Token lists for different networks
- [`src/assets/`](/src/assets/): Image assets for tokens
